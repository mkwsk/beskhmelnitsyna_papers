## 2. Назначение методики

САТ предназначен для оценки самоактуализации личности в традиции POI Шострома. В текущей теме может использоваться для измерения общей личностной самоактуализации, но не заменяет профессиональную самореализацию.

## 3. Теоретическое основание

- Основная теория: гуманистическая психология А. Маслоу, К. Роджерса, Э. Шострома.
- Ключевые понятия: ориентация во времени, поддержка, ценности, гибкость поведения, сензитивность, самоуважение, самопринятие и др.
- Значение для ВКР: может быть теоретически связан с самоактуализацией, но слишком широк для гипотезы о профессиональной самоактуализации.

## 6. Психометрические характеристики

В рабочем каталоге психометрические характеристики САТ не заполнены полностью. В анализе батареи отмечен риск по POI-традиции: для некоторых российских адаптаций семейства выявлялись проблемы надежности, валидности и дублирования пунктов. Поэтому при включении САТ нужно отдельно обосновать выбор и проверить требования кафедры.

## 9. Шкала ответов

Формат: в каждом пункте респондент выбирает одно из двух суждений, которое больше соответствует его взглядам или поведению.

## 10. Вопросы / утверждения

Файл с пунктами: `items/sat_self_actualization_test_ru_items.csv`

Полные пункты не внесены.

## 11. Ключи обработки

Каждый ответ, совпадающий с ключом шкалы, получает 1 балл. Затем баллы суммируются по шкалам и интерпретируются как профиль.

## 12. Интерпретация результатов

Интерпретировать как профиль самоактуализации. Для ВКР лучше заранее определить, какие шкалы теоретически связаны с гипотезой, но не сокращать методику вручную без обоснования.

## 15. Контроль качества данных

Особенно важно отслеживать время заполнения, пропуски и усталость респондента. При большом онлайн-опросе САТ может снизить завершение анкеты.

## 16. Ограничения методики

* Очень длинная для онлайн-батареи.
* Измеряет общую, а не профессиональную самоактуализацию.
* В POI-традиции есть методические вопросы по структуре и надежности.

## 17. Решение по включению в батарею

Итоговое решение: оставить как резерв / пересмотреть.

Обоснование: методика указана во введении, но для финальной батареи лучше рассмотреть более короткие и точные альтернативы.
