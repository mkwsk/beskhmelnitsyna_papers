# Каталог методик для ВКР по психологии

Дата сборки: 2026-06-16

Тема ВКР: взаимосвязь самооценки с уровнем профессиональной самоактуализации женщин-предпринимателей.

Этот файл собирает в одном месте методики, которые встречаются в доступных источниках проекта: во введении ВКР, списке ссылок, анализе батареи методик, статье по русскоязычной версии Core Self-Evaluation Scale и найденных первичных/вторичных источниках по альтернативным шкалам.

Полные тексты пунктов не везде продублированы. Для методик с неясным правовым статусом или источниками-перепечатками оставлена ссылка на источник и пометка `full_items_stored: false`. Это сделано специально, чтобы не превращать рабочий репозиторий ВКР в перепечатку тестовых материалов.

## Рекомендованная компактная батарея

Если в гипотезе остаются три переменные - самооценка, самоэффективность и профессиональная самоактуализация / самореализация, самый аккуратный набор выглядит так:

1. GSE - общая самоэффективность.
2. RSES Розенберга - классическая глобальная самооценка.
3. Методика Гавриловой - тип и уровень профессиональной самореализации.
4. Для личностной самоактуализации как дополнительного блока: SISA, Authenticity Scale или PGIS-II, но только после уточнения, какой именно аспект самоактуализации нужен в гипотезе.

CSEs(Ru) лучше не ставить одновременно с GSE без отдельной подгипотезы, потому что базовое самооценивание включает самоэффективность, самооценку, эмоциональную стабильность и локус контроля.
