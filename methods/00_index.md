# Каталог методик для ВКР по психологии

Дата актуализации: 2026-06-22

Тема ВКР: взаимосвязь самооценки с уровнем профессиональной самоактуализации женщин-предпринимателей.

Каталог обновлен по результатам аудита: карточки методик приведены к единому машинообрабатываемому шаблону `schema_version: "2.0"`. Человекочитаемое описание осталось в Markdown, а ключевые поля для скриптов вынесены во front matter и CSV-файлы.

Пункты методик хранятся в `methods/items/`, ключи подсчета - в `methods/keys/`. Активные методики имеют полный набор пунктов и ключей для сборки формы и автоматического подсчета.

Сжатого архива тестов в репозитории нет. Текущий архив тестов - это распределенная структура `methods/*.md`, `methods/items/*.csv`, `methods/keys/*.csv` и `forms/yandex/methods_manifest.json`.

## Что теперь можно парсить скриптами

- `schema_version`, `id`, `status_in_research`, `priority`, `item_count`, `response_format`, `form_variable_prefix` из YAML front matter.
- `response_options` и `scale_codes` из YAML front matter.
- CSV-карты пунктов из `methods/items/`.
- CSV-ключи подсчета из `methods/keys/`.
- `scoring_status`: использовать как предохранитель, чтобы скрипт не считал методики с непроверенным ключом.

## Методические замечания

САМОАЛ остается самым объемным блоком батареи. Его не нужно сокращать вручную: если методика включается, она должна переноситься в форму целиком и с ключом выбранного источника.

GSE и CSEs(Ru) частично пересекаются по смыслу, потому что базовое самооценивание включает самоэффективность как один из компонентов. В текущей версии они оставлены одновременно, потому что батарея задана куратором.

RSES и CSEs(Ru) нужно разводить в интерпретации: RSES используется как показатель глобальной самооценки / самоуважения, а CSEs(Ru) - как более широкий интегральный показатель базового самооценивания.

По аудиту ключей структурных ошибок в активных ключах не найдено. Для RSES ключ направлений оставлен без изменений, но отдельно зафиксирована конвенция шкалирования 1-4 и пересчет в конвенцию 0-3.
