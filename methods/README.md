# Каталог методик для ВКР по психологии

Дата актуализации: 2026-06-19

Тема ВКР: взаимосвязь самооценки с уровнем профессиональной самоактуализации женщин-предпринимателей.

Этот файл собирает методики, которые встречаются в материалах проекта. Актуальная батарея тестов обновлена по решению куратора.

Полные тексты пунктов не везде продублированы. Для методик с неясным правовым статусом или источниками-перепечатками оставлена ссылка на источник и пометка `full_items_stored: false`. Это сделано специально, чтобы не превращать рабочий репозиторий ВКР в перепечатку тестовых материалов.

## Методические замечания

САМОАЛ остается самым объемным блоком батареи. Его не нужно сокращать вручную: если методика включается, она должна переноситься в форму целиком и с ключом из выбранного источника.

GSE и CSEs(Ru) частично пересекаются по смыслу, потому что базовое самооценивание включает самоэффективность как один из компонентов. В текущей версии они оставлены одновременно, потому что батарея задана куратором.

RSES и CSEs(Ru) нужно разводить в интерпретации: RSES используется как показатель глобальной самооценки / самоуважения, а CSEs(Ru) - как более широкий интегральный показатель базового самооценивания.
