# Каталог методик для ВКР по психологии

Дата актуализации: 2026-06-22

Каталог обновлен по результатам аудита: карточки активных методик приведены к единому машинообрабатываемому шаблону `schema_version: "2.0"`. Человекочитаемое описание осталось в Markdown, а ключевые поля для скриптов вынесены во front matter и CSV-файлы.

Пункты методик хранятся в `methods/items/`, ключи подсчета - в `methods/keys/`. Неиспользуемые файлы-перенаправления удалены.

Отдельного сжатого архива тестов в текущей структуре нет. Под архивом тестов понимается логический markdown/CSV-набор: карточки методик, CSV с пунктами, CSV с ключами и манифест подключения формы.

## Что теперь можно парсить скриптами

- `schema_version`, `id`, `status_in_research`, `priority`, `item_count`, `response_format`, `form_variable_prefix` из YAML front matter.
- `response_options` и `scale_codes` из YAML front matter.
- CSV-карты пунктов из `methods/items/`.
- CSV-ключи подсчета из `methods/keys/`.
- `scoring_status` как предохранитель для автоматического подсчета.

## Методические замечания

САМОАЛ не сокращать вручную: если методика включается, она должна переноситься в форму целиком и с ключом выбранного источника.

GSE и CSEs(Ru) частично пересекаются по смыслу, потому что базовое самооценивание включает самоэффективность как один из компонентов.

RSES и CSEs(Ru) нужно разводить в интерпретации: RSES - глобальная самооценка / самоуважение, CSEs(Ru) - более широкий интегральный показатель базового самооценивания.

По аудиту ключей структурных ошибок в активных ключах не найдено. Главная оговорка относится к RSES: в репозитории используется кодирование ответов 1-4 и диапазон 10-40, а в части академических публикаций встречается кодирование 0-3 и диапазон 0-30. Для сопоставления использовать пересчет `rses_total_0_3 = rses_total_1_4 - 10`.

## Остальные методики

Гаврилова, САТ, SISA, Authenticity Scale, PGIS-II и MAS сохранены в архиве как reserve-кандидаты. Их ключи нужно сверить с первоисточниками перед автоматическим подключением к форме.
