### 11.2. Формулы подсчета

```text
score_q01 = raw_q01
score_q02 = reverse(raw_q02)

scale_1 = sum(score_q01, score_q02, ...)
scale_2 = sum(...)
total = scale_1 + scale_2 + ...
```

### 11.3. Работа с пропусками

Правило обработки пропусков:

```text
Если пропущено 0 пунктов - считать шкалу штатно.
Если пропущено до N пунктов в шкале - заменить средним по заполненным пунктам этой шкалы.
Если пропущено больше N пунктов - считать результат шкалы недействительным.
```

### 12.4. Что нельзя выводить по методике

* Методика не является клинической диагностикой.
