## 2. Назначение методики

Методика предназначена для определения типа профессиональной самореализации субъекта вне зависимости от возраста, стажа, образования, профессиональной области и карьерного уровня. В ВКР она может выступать ключевым инструментом для измерения профессиональной части самоактуализации женщин-предпринимателей.

## 3. Теоретическое основание

- Основная теория: деятельностный и субъектный подход к профессиональному развитию.
- Ключевые понятия: профессиональная самореализация, профессиональные цели, ресурсы, удовлетворенность деятельностью.
- Измеряемые компоненты:
  - ценностно-целевой компонент;
  - ресурсный компонент;
  - феноменологический компонент.
- Значение для ВКР: методика позволяет не просто оценить общую самоактуализацию, а связать ее именно с профессиональной деятельностью.

### 12.3. Типы профессиональной самореализации

Тип определяется по сочетанию выраженности трех компонентов Ц, Р и Ф. Порог: компонент считается выраженным при 23-34 баллах, невыраженным при 0-22 баллах.

| Тип | Ц | Р | Ф | Краткая интерпретация |
|---|---|---|---|---|
| Успешная профессиональная самореализация | + | + | + | Цели, ресурсы и субъективное переживание реализации согласованы |
| Прогнозируемая успешная | + | + | - | Есть цели и ресурсы, но пока нет переживания реализованности |
| Романтическая | + | - | + | Есть цели и переживание смысла, но не хватает ресурсов |
| Астеническая | - | + | + | Есть ресурсы и переживание включенности, но слабее целевая осознанность |
| Прогнозируемая ложная | + | - | - | Цели есть, но ресурсы и переживание реализации слабы |
| Нерефлексируемая / мнимая | - | + | - | Ресурсы есть, но цель и субъективная реализованность не выражены |
| Ложная | - | - | + | Есть субъективное переживание, но без целей и ресурсов |
| Формальное выполнение деятельности | - | - | - | Компоненты профессиональной самореализации не выражены |
