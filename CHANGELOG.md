# CHANGELOG

## 2026-06-19 - обязательные вопросы по страницам

- В `forms/yandex/scripts/create_form.py` добавлена принудительная обязательность для всех реальных вопросов при создании формы через API.
- Информационные блоки `comment` не помечаются как обязательные.
- Номер страницы теперь определяется порядком секций в `section_files`: одна секция равна одной странице / блоку формы.
- Это позволяет не переходить к следующему блоку без заполнения вопросов текущего блока.
